# PHPでのmessenger bot sample

## Requirements
* Heroku
* composer
* php

## 準備するもの

Heroku->Dashboard->Settings->Config Varsで

FACEBOOK_PAGE_ACCESS_TOKENとFACEBOOK_PAGE_VERIFY_TOKENを設定する

`composer`をインストール

Macの場合はhomebrewでいれる
```
$ brew install homebrew/php/composer
```

## 始め方

```
$ composer install
```
